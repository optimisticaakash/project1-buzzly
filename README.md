Buzzly is a modern full-stack social media web application where users can connect, share posts, create stories, follow people, and chat in real time.

## 📌 Features

### 🔐 Authentication
- Google Sign-in
- Email login
- Protected routes
- Session management using Clerk

### 🏠 Feed
- Create post (text + image)
- Like posts
- Real-time UI updates
- Responsive design

### 👤 Profile
- View own profile
- View other users' profiles
- Followers / Following count
- Edit profile
- Tabs (Posts / Media / Likes)

### 🤝 Connections
- Follow / Unfollow users
- Followers list
- Following list
- Pending requests

### 💬 Messaging
- One-to-one chat
- Message history
- Dynamic routing to user profile

### 📸 Stories
- Create text story
- Background color selection
- Story modal preview

### 🔍 Discover
- Search users by name / username / bio
- Follow directly from discover page

## 🧠 Tech Stack

### Frontend
- React.js
- React Router DOM
- Tailwind CSS
- Axios
- Lucide Icons

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

### Authentication
- Clerk

### Deployment
- Frontend → Vercel
- Backend → Render / Railway
- Database → MongoDB Atlas
